# Credit-Card-Default-Prediction

## Project Overview
This project focuses on predicting whether a credit card customer is likely to default on their payment using Machine Learning classification algorithms.

Banks and financial institutions issue credit cards to millions of customers, but one major challenge is identifying customers who may fail to repay their credit balance on time. This project aims to build predictive models that help minimize financial risk while maximizing profitability.

The dataset contains customer demographic details, payment history, bill statements, and previous credit information collected from credit card clients in Taiwan.

---

## Objective
The main objective of this project is to:

- Predict customer credibility
- Identify potential defaulters
- Reduce credit risk for banks
- Compare multiple classification algorithms
- Improve decision-making in the banking sector

---

## Dataset Information
The dataset includes:

- Demographic Information
- Credit Limit
- Gender
- Education
- Marital Status
- Age
- Repayment Status
- Bill Statements
- Previous Payment Amounts
- Default Payment Status

Target Variable:
- `Default Payment Next Month`
  - 1 = Default
  - 0 = Non-default
